pattern-0.1.0.0: A generalized representation of graph elements using category theory

Pattern is a recursive tree data structure that can be interpreted as different graph elements (nodes, relationships, subgraphs, paths) through categorical views. This is a reference implementation designed to be translatable to other languages while maintaining category-theoretic correctness.
